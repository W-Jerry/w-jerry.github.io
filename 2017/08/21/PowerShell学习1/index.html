PowerShell介绍

这里偷懒一下，可以通过PowerShell Wiki来了解PowerShell。

如何查看PowerShell版本

后面的学习至少需要3.0的版本，如果不是，请先升级到3.0。

PowerShell编辑器

现在有什么理由不使用Visual Studio Code，现在最新版本1.15.1。

好用的帮助系统

PowerShell包含了很多不同类型的可执行命令，其中主要的三种分别是：alias，function，cmdlet。

PowerShell提供了在线更新命令的功能update-help，我们应该养成定期更新的习惯。

我们可以使用man,help,get-help来获得帮助文档，这里为什么会有man ，有Linux和Unix经验的人应该不会陌生。

我们深入了解一下会发现，其实这3个命令是有本质的区别的，顺便让我们来看一下这个函数到底是什么。

通配符号*提供了了强大的匹配能力

如何你仔细查看命令的帮助文件，你会看到每一个命令参数的结尾都有[<CommonParameters>]这里泛指每个Cmdlet命令都可以使用一组包含8个参数(PowerShell 5.1)的集合。可以通过命令help about_CommonParameters

查看帮助。

小技巧

show command cmdlet可以使用图形化的方式将命令的参数展现出来。

PowerShell的扩展

PowerShell存在两种类型的扩展：module和PSSnapin。

PSSnapin基本已经被抛弃了，现在主要使用的都是module。

你会发现即使你没有使用import-module去加载模块，PowerShell依然可以自动发现模块中的命令。但是如果一个模块并不在PSModulePath环境变量中的话，就必须使用import-module
