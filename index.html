hexo安装的坑

Hexo报错，DEP0061

node.js从8.0开始就已经弃用了前面提到的fs.SyncWriteStream方法，然而作者项目里的一个叫hexo-fs的插件，调用了这个方法，所以就会报错，解决办法，作者没有提供，但是很显然，我只需要更新一下那个插件，就可以搞定，于是更新的方法是，在Hexo博客目录下，输入

error deploy not found: git

需要安装hexo-deployer-git
